# SecureVault

A cybersecurity-based client-host file transfer project developed using Python.  
This project enables secure communication and encrypted file sharing between a client and a host system.

# Features

- Secure file transfer
- Client-server architecture
- File encryption support
- User-friendly interface
- Fast and lightweight system
- Local network communication
- Organized project structure

# Technologies Used

- Python
- Socket Programming
- Encryption Libraries
- VS Code

# Security Notes

- Sensitive files are excluded using .gitignore 
- Encryption keys are not uploaded publicly.
- Intended for educational and learning purposes.

# Future Improvements
